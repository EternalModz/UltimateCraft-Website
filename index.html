Modding Software

Here you can find a collection of tools you can use to mod Minecraft on consoles.

Icon

PCK Studio - UltimateCraft Edition

This program can be used to create and modify .PCK files, which are used in skin packs, texture packs, and mashup packs; Otherwise known as DLC.

Icon

ARC Studio - UltimateCraft Edition

This program can be used to modify existing .ARC files, which are used in the display of strings, rendering of GUIs, and the loading of structure files. .ARC files also contain .FUI files, these are graphics and GUIs that can be edited by ARC Studio.

Icon

MSSSCMP Studio - UltimateCraft Edition

This program can be used to create and modify .MSSCMP files, which are archives containing sound files for actions such as eating and breaking blocks, and other audio.
